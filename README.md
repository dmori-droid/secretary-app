# 秘書アプリ

自分専用のシンプルな秘書アプリです。メモ・タスク管理・週次の振り返りを、ブラウザから操作できます。
Node.js の標準モジュールのみで動作し、追加のインストールは不要です。

## 機能

- **ダッシュボード**: メモ件数 / タスク件数 / 未完了タスク件数 / 最新の振り返り
- **メモ**: タイトルと本文で追加、一覧表示（作成日時）、削除
- **タスク**: 追加、一覧表示、完了/未完了の切り替え、削除
- **週次の振り返り**: 週ごとにコメントを記録（同じ週は上書き更新）、一覧表示、削除
- **保存**: データは `data.json` に保存され、アプリを閉じても消えません

## 使い方

```bash
node server.js
```

起動後、ブラウザで http://localhost:3000 を開きます。
終了するにはターミナルで `Ctrl + C` を押します。

## 構成

| ファイル | 役割 |
|---|---|
| `server.js` | Web サーバー + API（データの読み書き） |
| `public/index.html` | 画面（ダッシュボード・メモ・タスク・振り返り） |
| `package.json` | 起動設定 |
| `data.json` | 保存ファイル（自動生成／Git 管理対象外） |

## 備考

`data.json` には個人的なメモやタスクが入るため、`.gitignore` で Git 管理から除外しています。
初回起動時に自動生成されます。
